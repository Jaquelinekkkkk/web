Display dos elementos HTML

Os elementos dos html possuem exibição ou display inline ou block

BLOCK - é uma exibição que inicia na margem esquerda até a margem direita em que podemos determinar altura e largura do elemento. Por 33padrão, no HTML, elementos com exibição block não dividem espaço na horizontal com outros elementos.

INLINE - é uma exibição que permite que o elemento seja apresentado dentro de uma linha com outros elementos como um elemento textual.
