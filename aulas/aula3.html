CSS

CSS Externo
CCS externo é definido utilizando a tag link no início de documento HTML na sação de HEAD
< link rel="stylesheet" href="style.css" >
CSS incorporado tag style
O CSS incorpporado é definido comummente seção HEAD, podendo ser definido tambem em outros locais
css online
É definido por meio da propiedade style no proprio elçemento HTML no qual elçe saerá utilizado
