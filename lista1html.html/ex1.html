HTML

O objetivo dessa página é aprender os conceitos básicos de HTML.

A diferença entre a internet e a Web é que a internet é a rede global de computadores, enquanto a Web é um serviço que opera na internet, utilizando o protocolo HTTP para transmitir dados.

Sequência Sugerida de Leitura

  1. Início: Compreenda a base do projeto e a diferença entre Internet e Web.
  2. História da Web: Entenda a linha do tempo e os marcos históricos da rede.
  3. HTTP: Descubra como as requisições e respostas guiam a comunicação na rede.
  4. Currículo: Aplicação prática de tags de formatação em um perfil profissional.
  5. Contato: Interaja com um formulário estruturado de envio de dados.

Principais Conteúdos Estudados Até Agora

  • Estrutura básica do HTML (head, body, meta, title)
  • Tags de títulos (h1 a h6) e parágrafos (p)
  • Hiperlinks locais e externos usando a tag de âncora (a)
  • Listas ordenadas (ol), não ordenadas (ul) e de definição (dl)
  • Tabelas estruturadas (table, thead, tbody,tr, th, td)
  • Formulários com validação nativa (form, input, label)
